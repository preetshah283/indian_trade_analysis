India Trade Analysis

Project Overview

The India Trade Analysis project offers a detailed examination of India's trade activities, focusing on imports and exports. This analysis considers various dimensions such as country, commodity, section, year, region, GDP, growth, month, port, and trade intensity. Through data preprocessing, visualization, and advanced analytics, the project aims to reveal key patterns and trends in India's trade. The findings will be valuable for policymakers, businesses, and researchers. Additionally, the project includes a responsive web dashboard developed with React, featuring interactive insights, a detailed project description, Python code, and Jira integration for project management.
